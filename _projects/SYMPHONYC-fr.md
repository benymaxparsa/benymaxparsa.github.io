***Un projet concernant le cours de bases de données. Ce projet consiste en :***
- Collecte d'informations initiales pour la construction d'une base de données
- Modèle entité-relation amélioré
- Modèle relationnel
- Normalisation
- Implémentation de la base de données
- Requêtes SQL

## SYMPHONYC
* [Dépôt Github](https://github.com/benymaxparsa/SYMPHONYC)

SYMPHONYC est un projet de base de données complet conçu pour un service de streaming musical. Le projet démontre la conception et l'implémentation d'une base de données à partir de zéro, y compris la modélisation des données, la normalisation et les requêtes SQL.
